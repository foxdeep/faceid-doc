## 伺服器配置

1. 在 **Server IP** 欄位中輸入網址，在 **Port** 欄位中輸入埠。

範例:

```text
Server IP: http://192.168.8.27
Port: 8888
```

如果沒有 Port 可以只輸入網址:

```text
Server IP: http://foxconn.com
Port: (留空)
```

2. 輸入 **協力單元 Id** (不分大小寫) 此 id 來自[後台網址協力方授權](/docs/main/buddy-license)內建立後產生

   ![buddy](/assets/easyclockin/ch/buddy.png)

3. 防靜電測試器: 如果該裝置有連接防靜電測試器, 請下拉選單選擇對應的裝置, 如果沒有請選擇**無**

範例:

```text
協力單元名稱: Foxconn
協力單元 ID: B0D80C441
```

<img
  src={require('/assets/easyclockin/ch/setting-result.png').default}
  alt="setting-result"
  width="500"
/>

4. 按下**確認**後送出, 輸入皆正確會進入主要頁面

<img
  src={require('/assets/easyclockin/ch/home.png').default}
  alt="home"
  width="500"
/>

5. 目前有兩種模式 **出勤模組**和**ESD 模組**

ESD 模組是專為防靜電手環測試設計，確保配戴防靜電手環的人員已正確接地。此模組可用來驗證 ESD 防護措施的有效性，適用於敏感的生產或作業環境。

## 上傳照片

<img
  src={require('/assets/easyclockin/ch/reupload.png').default}
  alt="reupload"
  width="500"
/>

如何上傳照片:

1. 點擊 **設定**
2. 點擊 **上傳**
3. 輸入 **工號**
4. 輸入後台設定的 **生日 (MM-DD)**
5. 按下 **確認** 後會倒數 3 秒對準框框進行拍攝

   <img
     src={require('/assets/easyclockin/ch/confirm.png').default}
     alt="confirm"
     width="500"
   />

6. 點擊 **重拍** 可重新拍照。
7. 點擊 **上傳** 上傳照片。

**上傳** 成功或失敗都會顯示訊息, 上傳的照片可於後台觀看

## 出勤模組-感應卡片

需要在[後台](/docs/attendance/activated-team#%E9%A1%AF%E7%A4%BA%E7%9A%84%E9%97%9C%E9%8D%B5%E4%BF%A1%E6%81%AF)設定 **人臉辨識+員工卡** 為打卡模式

<img
  src={require('/assets/easyclockin/ch/RFID.png').default}
  alt="RFID"
  width="500"
/>

### 畫面資訊

1. 顯示 **協力單元名稱**
2. 顯示 **版本**
3. 顯示 **目前日期時間**

### 感應步驟

感應 **感應卡片** 成功後會自動跳轉到 **臉部辨識** 畫面

如果 **3 至 4 秒** 內沒有偵測到臉部，會自動跳回 **感應卡片** 頁面

### 感應失敗

<img
  src={require('/assets/easyclockin/ch/RFID-fail.png').default}
  alt="RFID-fail"
  width="500"
/>

如果感應失敗，請重新感應卡片。

## ESD 模組

有三個模式 **人臉辨識+ESD**, **員工卡+ESD**, **人臉辨識+員工卡+ESD**

上方三個 icon 為導引列, 由左而右依序表示為 **員工卡** > **人臉辨識** > **ESD**, 用來告知使用者目前操作流程
紅色 icon 代表進行中尚未完成操作流程, 綠色 icon 代表已完成這個操作流程

### 人臉辨識+員工卡+ESD

<img
  src={require('/assets/easyclockin/ch/esd-main.png').default}
  alt="esd-main"
  width="500"
/>

感應完卡片後, 會跳轉到 **臉部辨識** 畫面

<img
  src={require('/assets/easyclockin/ch/esd-main2.png').default}
  alt="esd-main2"
  width="500"

<img
  src={require('/assets/easyclockin/ch/esd-main3.png').default}

1. 顯示使用者資訊
2. 頁面倒數秒數(可在後台設定), 倒數完畢會關閉此頁面回到第一個操作流程步驟
3. 顯示目前 ESD 操作結果
4. 顯示目前 ESD 操作步驟
5. 顯示目前 ESD 偵測到的數值
6. 手動取消會關閉此頁面回到第一個操作流程步驟

如果數值沒有通過：

<img

## 管理者設定頁面

<img
  src={require('/assets/easyclockin/ch/hide-settings.png').default}
  alt="hide-settings"
  width="500"
/>

1. 點選右上角 **協力單元名稱**
2. 下方會跳出設定 icon, 點擊該 icon

<img
  src={require('/assets/easyclockin/ch/hide-settings-password.png').default}
  alt="hide-settings-password"
  width="500"
/>

為了避免普通員工不誤觸, 需要管理者輸入密碼, 密碼為**2317**

<img
  src={require('/assets/easyclockin/ch/hide-settings-list.png').default}
  alt="hide-settings-list"
  width="500"
/>

1. 重新設定伺服器位址, 需要點選右邊**重新啟動**, app 會重啟才會生效
2. 顯示目前安裝的系統版本號
3. 如果發生問題可以點選**上傳紀錄**, 會將目前的裝置紀錄的資訊上傳到後台
4. 重新選擇對應的 ESD 裝置
5. 點擊**登出**按鈕, 會回到初始設定頁面
