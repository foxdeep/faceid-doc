### 概述

**協力方授權** 頁面提供設備授權的管理概覽，允許用戶查看、搜尋、篩選及管理授權資訊。

### 顯示的關鍵資訊

- **狀態**：顯示授權是否 **在線** 或 **離線**。
- **ID**：每個協力單元的唯一識別碼。
- **版本**：授權設備所使用的軟體版本。
- **名稱**：為協力單元設定的識別名稱。
- **識別群組**：顯示授權所屬的 **群組** 或 **虛擬群組**。
- **創建時間**：授權的生成時間戳。

---

### 創建協力單元

要新增協力單元，請按照以下步驟操作：

1. 進入 **協力方授權** 頁面。
2. 點擊 **創建協力單元** 按鈕。
3. 輸入協力單元 的 **名稱**。
4. 選擇綁定的 **群組** 或 **虛擬群組** 以進行識別。
5. 確認並保存設置。
