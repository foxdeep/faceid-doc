### 概述

**用戶管理**頁面允許管理員管理用戶帳戶和角色。

> 此頁面僅對具有**管理員**角色的用戶開放。

---

### 創建用戶

![user](/assets/main/user.png)

創建用戶的步驟：

1. 點擊位於畫面右下角的**+**按鈕。
2. 輸入用戶的詳細信息，並確保密碼符合要求。
3. 確認並保存用戶設置。

### 管理用戶

- **權限設定**：點擊用戶旁邊的**更多菜單**來調整權限。
- **刪除用戶**：將用戶從系統中移除（此操作不可逆）。
