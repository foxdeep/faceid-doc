## 服务器配置

1. 在 **Server IP** 栏位中输入网址，在 **Port** 栏位中输入端口。

示例：

```text
Server IP: http://192.168.8.27
Port: 8888
```

如果没有端口可以只输入网址：

```text
Server IP: http://foxconn.com
Port: (留空)
```

2. 输入 **协力单元 Id**（不区分大小写），此 ID 来自[后台网址协力方授权](/docs/main/buddy-license)内创建后生成。

   ![buddy](/assets/easyclockin/ch/buddy.png)

3. 防静电测试器: 如果该设备连接了防静电测试器，请在下拉菜单中选择对应的设备，如果没有请选择 **无**。

示例：

```text
协力单元名称: Foxconn
协力单元 ID: B0D80C441
```

<img
  src={require('/assets/easyclockin/ch/setting-result.png').default}
  alt="setting-result"
  width="500"
/>

4. 按下 **确认** 后提交，输入正确会进入主页面。

<img
  src={require('/assets/easyclockin/ch/home.png').default}
  alt="home"
  width="500"
/>

5. 目前有两种模式 **出勤模块** 和 **ESD 模块**

ESD 模块专为防静电手环测试而设计，确保佩戴防静电手环的人员已正确接地。该模块用于验证 ESD 防护措施的有效性，适用于敏感的生产或作业环境。

## 上传照片

<img
  src={require('/assets/easyclockin/ch/reupload.png').default}
  alt="reupload"
  width="500"
/>

如何上传照片：

1. 点击 **设置**
2. 点击 **上传**
3. 输入 **工号**
4. 输入后台设置的 **生日 (MM-DD)**
5. 按下 **确认** 后会倒数 3 秒对准框框进行拍摄

   <img
     src={require('/assets/easyclockin/ch/confirm.png').default}
     alt="confirm"
     width="500"
   />

6. 点击 **重拍** 可重新拍照。
7. 点击 **上传** 上传照片。

**上传** 成功或失败都会显示消息，上传的照片可于后台查看。

## 出勤模块 - 感应卡片

需要在[后台](/docs/attendance/activated-team#%E6%98%BE%E7%A4%BA%E7%9A%84%E5%85%B3%E9%94%AE%E4%BF%A1%E6%81%AF)设置 **人脸识别+身份卡** 为打卡模式

<img
  src={require('/assets/easyclockin/ch/RFID.png').default}
  alt="RFID"
  width="500"
/>

### 画面信息

1. 显示 **协力单元名称**
2. 显示 **版本**
3. 显示 **当前日期时间**

### 感应步骤

感应 **感应卡片** 成功后会自动跳转到 **人脸识别** 画面

如果 **3 至 4 秒** 内没有检测到人脸，会自动跳回 **感应卡片** 页面

### 感应失败

<img
  src={require('/assets/easyclockin/ch/RFID-fail.png').default}
  alt="RFID-fail"
  width="500"
/>

如果感应失败，请重新感应卡片。

## ESD 模块

有三种模式 **人脸识别+ESD**、**身份证+ESD**、**人脸识别+身份证+ESD**

上方三个图标为导航栏，从左到右依次表示为 **身份证** > **人脸识别** > **ESD**，用来告知用户当前操作流程
红色图标代表进行中尚未完成操作流程，绿色图标代表已完成该操作流程

### 人脸识别+身份证+ESD

5. 点击 **登出** 按钮，会回到初始设置页面
