### 概述

用户可以在 **下载** 页面下载 ESD 模组的 APK。该 APK 与出勤模组使用的 APK 相同，但运行在特定于 ESD 合规的 **伙伴模式** 下。

- **下载**: 点击 **下载** 按钮下载 APK。
- **二维码**: 扫描二维码下载 APK。
- **版本**: 显示当前 APK 版本。
- **上传日期**: 显示 APK 的上传日期。

**注意:** 请确保系统已配置为 ESD 专用的伙伴模式，以启用 ESD 跟踪功能。
