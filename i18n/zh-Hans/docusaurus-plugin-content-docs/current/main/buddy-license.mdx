### 概述

**协力方授权** 页面提供设备授权的管理概览，允许用户查看、搜索、筛选及管理授权信息。

### 显示的关键信息

- **状态**：显示授权是否 **在线** 或 **离线**。
- **ID**：每个协力单元的唯一识别码。
- **版本**：授权设备所使用的软件版本。
- **名称**：为协力单元设置的识别名称。
- **识别群组**：显示授权所属的 **群组** 或 **虚拟群组**。
- **创建时间**：授权的生成时间戳。

---

### 创建协力单元

要新增协力单元，请按照以下步骤操作：

1. 进入 **协力方授权** 页面。
2. 点击 **创建协力单元** 按钮。
3. 输入协力单元 的 **名称**。
4. 选择绑定的 **群组** 或 **虚拟群组** 以进行识别。
5. 确认并保存设置。
