### Overview

The **User Management** page allows administrators to manage user accounts and roles.

> This page only open to users with the **Admin** role.

---

### Creating a User

![user](/assets/main/user.png)

To create a User:

1. Click the **+** button located at the bottom-right corner of the screen.
2. Enter the user's details, ensuring the password meets the specified requirements.
3. Confirm and save the User setup.

### Managing a User

- **Modify Permissions**: Click the three-dot menu next to a user to adjust permission settings.
- **Delete Users**: Remove users from the system (irreversible action).
