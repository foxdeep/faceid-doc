# EasyClockIn

## Installation

After installation, the screen will look like the image below:

<img src="/assets/easyclockin/en/settings.png" alt="settings" width="500" />

## Server Configuration

1. Enter the URL in the **Server IP** field, and the port in the **Port** field.

Example:

```text
Server IP: http://192.168.8.27
Port: 8888
```

If there is no port, just enter the URL:

```text
Server IP: http://foxconn.com
Port: (leave empty)
```

2. Enter the **Buddy Id** (case-insensitive). This ID is generated after being created in the **Third-party Authorization** section of the backend.

![buddy](/assets/easyclockin/en/buddy.png)

3. **ESD** is the static electricity tester. If the device is connected to a static electricity tester, please select the corresponding device from the dropdown menu. If not, please select **NONE**.

Example:

```text
Buddy name: Foxconn
Buddy ID: B0D80C441
```

<img
  src="/assets/easyclockin/en/setting-result.png"
  alt="setting-result"
  width="500"
/>

4. After clicking **Confirm**, if all information is correct, you will be directed to the main page.

<img src="/assets/easyclockin/en/home.png" alt="home" width="500" />

5. Currently, there are two modes: **Attendance Module** and **ESD Module**.

## Upload Photo

<img src="/assets/easyclockin/en/reupload.png" alt="reupload" width="500" />

To re-upload a photo:

1. Click **Settings**.
2. Click **Upload**.
3. Enter the **Employee ID**.
4. Enter the **Birthday (MM-DD)** as set in the backend.
5. Click **Confirm** to start a 3-second countdown and take a photo.

   <img src="/assets/easyclockin/en/confirm.png" alt="confirm" width="500" />

6. Click **Again** to retake the photo if necessary.
7. Click **Upload** to upload the photo.

A success or failure message will be displayed, and the uploaded photo can be viewed in the backend.

## Attendance module - RFID Card

You need to set up both facial recognition and ID card in the [backend](/docs/attendance/activated-team#key-information-displayed).

<img src="/assets/easyclockin/en/RFID.png" alt="RFID" width="500" />

### Display Information

1. Display the third-party unit name
2. Display the URL and version
3. Display the current date and time

### Scanning Process

After successfully scanning the **RFID card**, the system will enter the **Facial Recognition** page.

If no face is detected within **3 to 4 seconds**, the system will automatically return to the **RFID card** page.

### Scan Failure

<img src="/assets/easyclockin/en/RFID-fail.png" alt="RFID-fail" width="500" />

If the scan fails, retry scanning the card.

## ESD Module

There are three modes: Face Recognition + ESD, ID Card + ESD, and Face Recognition + ID Card + ESD.

The three icons at the top serve as a navigation bar, representing ID Card > Face Recognition > ESD from left to right, indicating the current step in the process. A red icon means the step is in progress and not yet completed; a green icon means the step is completed.

### Face Recognition + ID Card + ESD

<img src="/assets/easyclockin/en/esd-main.png" alt="esd-main" width="500" />

After sensing the card, the interface switches to the Face Recognition screen.

<img src="/assets/easyclockin/en/esd-main2.png" alt="esd-main2" width="500" />

After successful recognition, it switches to the ESD screen.

<img src="/assets/easyclockin/en/esd-main3.png" alt="esd-main3" width="500" />

1. Displays user information
2. Page countdown timer (configurable in the backend); when the countdown ends, the page closes and returns to the first step
3. Shows the current ESD operation result
4. Shows the current ESD operation step
5. Shows the current value detected by ESD
6. Manual cancel will close the page and return to the first step

If the value does not pass:

<img src="/assets/easyclockin/en/esd-main4.png" alt="esd-main4" width="500" />

If the value passes, the page will close and return to the first step after waiting for 1 second:

<img src="/assets/easyclockin/en/esd-main5.png" alt="esd-main5" width="500" />

If the device is not connected to ESD, this screen will appear. Check if the ESD name is correct. If not, you can click the Logout button to return to the initial settings page.

## Administrator Settings Page

<img
  src="/assets/easyclockin/en/hide-settings.png"
  alt="hide-settings"
  width="500"
/>

1. Tap the unit name at the top right
2. A settings icon will appear at the bottom; tap the icon

<img
  src="/assets/easyclockin/en/hide-settings-password.png"
  alt="hide-settings-password"
  width="500"
/>

To prevent accidental access by regular employees, administrators need to enter the password, which is 2317.

<img
  src="/assets/easyclockin/en/hide-settings-list.png"
  alt="hide-settings-list"
  width="500"
/>

1. Reset the server address (tap Restart on the right; the app will reboot for changes to take effect)
2. Display the currently installed system version number
3. If an issue occurs, tap Upload Logs to upload the device logs to the backend
4. Re-select the corresponding ESD device
5. Tap the Logout button to return to the initial settings page
