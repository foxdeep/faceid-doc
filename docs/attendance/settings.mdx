### Overview

The Attendance module’s settings page allows you to customize its functionality. The settings are organized into two main sections:

- **Purchase Plan**: This section displays the purchased plan for the Attendance module.
- **Preferences**: Here, you can configure the module’s preferences, including the activated teams. It’s essential to configure these preferences before using the Attendance module.

**Note**: The settings page is exclusively accessible to the admin role.
