### Overview

Users can download the EasyClockIn apk in the **Download** page.

- **Download**: Click the **Download** button to download the EasyClockIn apk.
- **QR Code**: Scan the QR code to download the EasyClockIn apk.
- **Version**: The current version of the EasyClockIn apk.
- **Upload Date**: The date when the EasyClockIn apk was uploaded.
