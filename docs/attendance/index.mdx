The Attendance Module in OWL FaceID Cloud Service offers a thorough system for tracking and managing attendance records. It features several tools aimed at simplifying scheduling, record-keeping, and identity verification.

### Features

The Attendance Module includes several essential components:

- **Dashboard** – Gives a quick look at attendance data, with real-time statistics and reports.
- **Activated Teams** – Shows the teams currently using the attendance module.
- **ID Cards** – Handles digital ID cards for identification within the system.
- **Temporary Cards** – Enables the creation of temporary access cards for visitors or short-term members.
- **Time Card** – Shows clock-in and clock-out records for members.
- **Meeting** – Schedules and tracks attendance for meetings.
- **Realtime Events** – Keeps an eye on live recognition events.
- **Download** – Offers the EasyClockIn APK for easy access.
- **Settings** – Allows you to customize attendance parameters and configurations.
