The ESD Module in OWL FaceID Cloud Service provides a comprehensive system for tracking and managing ESD compliance. It ensures that all personnel adhere to ESD safety protocols by integrating identity verification with ESD testing.

### Features

The ESD Module includes several essential components:

- **Dashboard** – Provides an overview of ESD compliance data, with real-time statistics and reports.
- **Activated Teams** – Displays the teams currently enrolled in ESD tracking.
- **ID Cards** – Manages digital ID cards for use in ESD testing.
- **Temporary Cards** – Enables the creation of temporary access cards for visitors or short-term members.
- **Events** – Displays detailed records of ESD-related events, including test results and member details.
- **Download** – Offers the APK for the ESD module.
- **Settings** – Allows you to customize ESD parameters and configurations.
