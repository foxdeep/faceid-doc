### Overview

The **Activated Teams** page lists all teams currently enrolled in ESD tracking. Users can:

- Check the status of each team and their buddies.
- Click on the buddy icon to view their details.

**Note:** Activated Teams must be selected in the [**Settings**](/docs/esd/settings) page to begin tracking ESD compliance.

### Buddies Management

![buddies](/assets/esd/en/buddies.png)

This dialog provides an overview of buddies for connected devices.

#### Key Information Displayed:

- **Status**: Indicates whether a license is **Online** or **Offline**.
- **ID**: Unique identifier for each Buddy License.
- **Version**: Software version associated with the license.
- **Name**: Assigned name for easy identification.
- **Work Mode**: Indicates whether a buddy is using one of the following modes:

  - **Face Recognition + ID Card + ESD**
  - **Face Recognition + ESD**
  - **ID Card + ESD**

  Users can modify the mode by clicking on the **three-dot menu** and selecting **Function Settings**.

  ![Function Settings](/assets/esd/en/function-settings.png)

- **Create Time**: Timestamp indicating when the license was generated.
