### Overview

Users can download the APK for the ESD module in the **Download** page. The APK is the same as the one used for the Attendance module, but it operates in **Buddy Mode** specific to ESD compliance.

- **Download**: Click the **Download** button to download the APK.
- **QR Code**: Scan the QR code to download the APK.
- **Version**: The current version of the APK.
- **Upload Date**: The date when the APK was uploaded.

**Note:** Ensure that the system is configured for ESD-specific Buddy Mode to enable ESD tracking features.
